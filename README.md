<h1 align="center">SMARTCINNAMON</h1>
<p align="center">
  <b>A Multi-Modal Machine Learning and IoT Framework for Precision Cinnamon Cultivation</b>
</p>

<hr/>

<h2>Project Overview</h2>
<p>
  SMARTCINNAMON is a comprehensive, team-based final year project designed to modernize Sri Lankaâ€™s
  cinnamon cultivation using Artificial Intelligence (AI), Machine Learning (ML), Internet of Things (IoT),
  and GIS technologies.
</p>
<p>
  The project addresses critical challenges across the cinnamon lifecycle, including soil nutrient management,
  disease prediction, plantation land utilization, and post-harvest quality preservation. By integrating
  multiple system capabilities into a single platform, SMARTCINNAMON enables data-driven decision-making
  for farmers, warehouse managers, and exporters.
</p>

<hr/>

<h2>Project Objectives</h2>
<ul>
  <li>Improve cinnamon yield and quality through precision agriculture techniques</li>
  <li>Enable early prediction and prevention of fungal diseases</li>
  <li>Optimize fertilizer usage based on real-time soil data</li>
  <li>Detect underutilized plantation land and forecast yield</li>
  <li>Ensure post-harvest quality compliance for export standards</li>
</ul>

<hr/>

<h2>Integrated System Scope</h2>
<p>SMARTCINNAMON provides an end-to-end solution across the cinnamon value chain:</p>
<ul>
  <li>Real-time soil and environmental monitoring using IoT</li>
  <li>Predictive analytics for plant disease progression</li>
  <li>Data-driven fertilizer recommendations tailored for cinnamon cultivation</li>
  <li>GIS-based vacant area detection and yield forecasting</li>
  <li>Post-harvest warehouse monitoring with risk alerts and corrective guidance</li>
  <li>Web and mobile dashboards for visualization, insights, and notifications</li>
</ul>

<hr/>

<h2>System Architecture</h2>
<p>The system follows a layered architecture:</p>
<ul>
  <li><b>Data Collection Layer:</b> IoT sensors, mobile image capture, satellite/drone imagery</li>
  <li><b>Processing &amp; Intelligence Layer:</b> Machine learning models, image processing, GIS analysis</li>
  <li><b>Application Layer:</b> Web and mobile dashboards</li>
  <li><b>Cloud Layer:</b> Data storage, analytics, alerts, and visualization</li>
</ul>
